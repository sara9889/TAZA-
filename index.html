ESP32 es la denominación de una familia de chips SoC de bajo costo y consumo de energía, con tecnología Wi-Fi y Bluetooth de modo dual integrada

Características

  • Características
  • Es compatible Arduino con el pluggin adecuado
  • Velocidad de reloj : Entre 160 Mhz y 240 Mhz
  • 520 Kb de RAM.
  • Wifi integrado
  • Bluetooth 4.2 2.4 Ghz; BT 2.0 y 4.0 BLE.
  • 36 GPIO pins.

La función que cumplirá el ESP32 es la de conectar nuestro circuito mediante Wi-FI
Arduino es una plataforma de creación de electrónica de código abierto, la cual está basada en hardware y software libre, flexible y fácil de utilizar.

  • Tiene 14 pines de entrada/salida de voltaje
  • Contienen puerto usb

  • El mismo hardware puede emplearse una y otra vez para infinidad de aplicaciones con sólo modificar las conexiones y la programación.

PROCESADOR

La protoboard es una placa de pruebas para electrónica que contiene numerosos orificios en los que es posible insertar cables y otros elementos electrónicos para montar circuitos provisionales.

Características:

  • Sirve para probar el correcto funcionamiento del circuito y depurarle las posibles fallas
  • No requiere soldar sus componentes para tener un circuito operativo.
  • La función que cumplirá es modificar los circuitos con la mayor rapidez para llevar acabo en el proyecto y montar el circuito de forma definitiva.

    • Los cables del protoboard son cables con un conector a cada lado se usa normalmente para interconectar entre sí los componentes en una placa de pruebas.

Características

  • Resistencia: 0.05 ohm por metro.
  • Material: Cobre estañado.
  • Calibre: 22 AWG.
  • Diámetro externo: 1.5 mm.
  • Diámetro del conductor: 0.6 mm.
  • Temperatura de operación: Hasta 110 °C
  • Material del aislante: PVC.
  • El rol que desempeña en nuestro proyecto es el de conectar los elementos con el protoboard

SALIDA

Esta matriz tiene 256 LED ultrabrillantes y su respaldo es de PCB grueso y flexible, gracias a esto se puede doblar y curvar suavemente alrededor de las superficies

Características

  • Se debe mantener el consumo de corriente por debajo de 5A,.
  • Se puede controlar los 256 LED con un solo pin de microcontrolador
  • Se puede configurar cada LED como se desee para desplazar mensajes o dibujar pequeñas imágenes
  • Esta matriz se puede usar como cualquier otro producto compatible con NeoPixel .
